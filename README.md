## 🧠 프로젝트 소개

이 프로젝트는 **p5.js**와 **ml5.js**의 `handPose` 모델을 사용하여  
**손 제스처로 그림을 그리고, 색상을 선택하고, 효과를 적용하는** 인터랙티브한 웹 애플리케이션입니다.

웹캠을 통해 손을 실시간으로 인식하며,  
엄지와 검지를 붙인 OK 제스처로 그림을 그리고,  
컬러 휠을 통해 색상을 바꾸고, 다양한 효과를 적용할 수 있습니다.

---

## 📦 주요 기능 요약

- ✍️ **OK 제스처로 그림 그리기**
- 🧼 **지우기 모드**: 손가락으로 특정 선을 지울 수 있음
- 🗑 **전체 그림 삭제 기능**
- 🎨 **HSV 컬러 휠**: 원하는 색상 선택 가능
- **다양한 효과 버튼** GOOD, FIREWORK, ❤️ HEART, 😢 SAD 버튼
- 📷 실시간 비디오 피드 기반 손 추적

---

## 📸 기술 스택

- `p5.js` – 그래픽 및 UI 구성
- `ml5.js` – 손 추적용 `handPose` 모델
- `JavaScript` – 전체 로직 제어

---

## ✅ 제스처 커맨드 안내

### ✍️ OK 제스처로 선 그리기
- **제스처**: 엄지와 검지를 붙여 OK 모양 만들기  
- **조건**: 쓰기 모드(Writing Mode)가 ON일 때만 동작  
- **동작**: 손가락 이동 경로에 따라 선이 실시간으로 그림

### 🧼 지우기 제스처
- **제스처**: 검지 손가락만 내민 상태  
- **조건**: Erase Mode가 ON일 때  
- **동작**: 손가락 주변의 선분을 제거 (약 25px 반경)

### 👐 양손 감지
- **제스처**: 두 손을 화면에 올리기  
- **동작**:
  - 현재 그리던 선 저장 및 초기화
  - 컬러 선택 중이면 색상 확정
  - “✔ 색상이 고정되었습니다” 메시지 출력

---

## 🎨 HSV 컬러 선택

- **컬러 휠**: 중심에서의 각도 → Hue, 반지름 거리 → Saturation  
- **색상 선택**: 손가락을 원하는 위치에 가져다 놓기  
- **확인**: 오른쪽 "확인" 버튼 위에 손가락 위치시키면 적용  
- **양손 감지 시** 색상 고정 및 확인 메시지 출력

---

## 🧩 버튼 구성 및 기능

| 버튼 이름 | 위치 | 기능 |

## ✨ 주요 기능

### 1. OK 제스처 드로잉
- 엄지와 검지를 붙인 OK 모양 → 선 그리기 시작
- 쓰기 모드(Writing Mode)에서만 동작

### 2. 지우기 모드
- Erase Mode를 켜면, 손가락이 지나간 경로의 선분이 삭제됨

### 3. HSV 컬러 휠로 색상 선택
- Color 버튼 → HSV 색상 선택 UI 등장
- 손가락으로 원하는 색 클릭 → 손 두 개 인식되면 색상 고정됨

### 4. 다양한 효과 버튼
- "GOOD", "FIREWORK, sad , love " 버튼 → 화면 중앙에 시각적 효과 표시

### 5. UI 버튼
- 쓰기 모드 ON/OFF
- 지우기 모드 ON/OFF
- 전체 선 지우기
- 컬러 휠 열기
- 이펙트 실행 (Good / Firework / Love / Sad)

---

## 🖐️ 제스처 커맨드 정리

| 손 제스처 | 동작 |
|-----------|------|
| **OK 제스처 (검지+엄지)** | 쓰기 모드에서 선 그리기 시작 |
| **검지만 펼침** | 지우기 모드에서 해당 위치 근처 선 삭제 |
| **양손 감지** | 현재 선 저장 / 컬러 선택 확정 |
| **단일 손 인식 중** | HSV 휠에서 색상 선택 / 버튼 클릭 / 그리기 가능 |

---

## 🎨 HSV 컬러 선택 UI

- 컬러 휠은 H(색상), S(채도), V(명도) 기반
- 색상 선택 후 오른쪽 "확인" 버튼에 손가락을 위치시키면 색 적용
- 양손 감지되면 "✔ 색상이 고정되었습니다" 문구 표시



## 🚀 사용 방법

1. `index.html`을 웹 브라우저에서 실행
2. 웹캠 허용 → 손 인식 시작
3. **쓰기 모드** ON → OK 제스처로 선 그리기
4. **컬러 버튼** → HSV 휠에서 색상 선택
5. **양손 감지 시 색상 고정**
6. 필요 시 **GOOD / FIREWORK / SAD / LOVE 버튼 클릭** → 화면 이펙트 발생
7. 전체 지우기, 지우기 모드 등 다양한 기능 사용 가능
